                        VIDEO İŞLEME AKIŞI (11 ADIM)
════════════════════════════════════════════════════════════════════════════════

INPUT: Video Dosyası
  │
  ├─→ [1] FFmpeg → Frame & Ses Çıkarma
  │
  ├─→ [2] YOLOv8 → Nesne Algılama
  │        (Araç, insan, bina vb. tespit)
  │
  ├─→ [3] Google Vision API → Landmark Algılama
  │        (Eiffel Kulesi, Roma Kolossiyumu, vb.)
  │
  ├─→ [4] Whisper → Ses Transkripsiyonu
  │        (Video sesini Türkçe metne çevir)
  │
  ├─→ [5] EasyOCR → Metni Çıkarma
  │        (Videodaki yazıları okuma)
  │
  ├─→ [6] NER (Turkish BERT) → Konum Çıkarma
  │        (Istanbul, Eiffel Kulesi, Blue Mosque vb.)
  │
  ├─→ [7] Nominatim → Geocoding
  │        (Konum adı → lat/lng koordinatları)
  │
  ├─→ [8] Location Deduplicator → Kopya Konum Kaldırma
  │        (5km içinde çiftleri birleştir)
  │
  └─→ [9] Route Optimizer → En İyi Rotayı Bul
           (TSP algoritması - komşu heuristic)
  │
  │ VERİTABANINA KAYDET
  │

                        API ENDPOINTS ÖZETI
════════════════════════════════════════════════════════════════════════════════

CORE-API (Port 8000)
────────────────────────────────────────────────────────────────────────────
GET   /
      → Hizmet durumu kontrol
      Response: {service, message, status}

GET   /health
      → Sağlık kontrolü
      Response: {status, service}

POST  /internal/videos/process
      → Video yükleme ve işleme başlatma
      Payload: multipart/form-data (video dosyası)
      Response: {video_id, status, message}

GET   /internal/videos/{video_id}
      → İşlenen video sonuçlarını alma
      Response: {Tüm AI analiz sonuçları}

────────────────────────────────────────────────────────────────────────────

MOBILE-BFF (Port 8001)
────────────────────────────────────────────────────────────────────────────
POST  /api/mobile/videos/upload
      → Mobil uygulamadan video yükleme
      Validasyon: MIME type, Max 100MB
      Proxy: Core-API

GET   /api/mobile/videos/{video_id}
      → Mobil cihaz için optimize sonuç
      Proxy: Core-API

────────────────────────────────────────────────────────────────────────────

WEB-BFF (Port 8002)
────────────────────────────────────────────────────────────────────────────
GET   /health
      → Sağlık kontrolü
      (Diğer endpoints geliştirme aşamasında)

                        KULLANILAN TEKNOLOJİLER
════════════════════════════════════════════════════════════════════════════════

Backend Framework:
  • FastAPI (Python web framework)
  • SQLAlchemy (ORM)
  • Pydantic (Data validation)

AI/ML Services:
  • YOLOv8 (Object detection)
  • Google Cloud Vision API (Landmark detection)
  • OpenAI Whisper (Speech-to-text)
  • HuggingFace Transformers (Turkish BERT for NER)
  • EasyOCR (Optical character recognition)
  • Nominatim/OpenStreetMap (Geocoding)

Databases:
  • PostgreSQL (Relational data)
  • Redis (Caching + Rate limiting)
  • MongoDB (Document storage - optional)

Infrastructure:
  • Docker & Docker Compose
  • FFmpeg (Video/audio processing)
  • Celery (Task queue - not active)
